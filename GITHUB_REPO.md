## About

```text
Python Telegram bot built with aiogram and yt-dlp for downloading supported Instagram, YouTube, YouTube Shorts, and SoundCloud media. Includes Persian, English, Arabic, and German UI, admin activation, public access toggles, per-user and global cookies.txt upload, forced channel subscription, safe long-caption handling, SoundCloud cover/audio delivery, album sending for multi-item Instagram posts, and simple Ubuntu install/update/remove scripts.
```
