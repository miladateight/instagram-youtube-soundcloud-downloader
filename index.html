Media Downloader Bot · AT8 project

A private Telegram bot for YouTube, Instagram, and SoundCloud

A self-hosted Telegram bot, written entirely in Python, that downloads supported media on request. It stays inactive after installation until the admin enables it from inside Telegram, and ships with its own Ubuntu installer and systemd service.

The problem

Saving a video, reel, or track you were sent usually means juggling separate downloader sites and apps, most of which are ad-heavy, unreliable, or simply stop working when a platform changes its pages.

What this bot does

  • Detects YouTube, YouTube Shorts, Instagram, and SoundCloud links automatically.
  • Delivers video, photo, audio, or document through yt-dlp, in Telegram-friendly formats.
  • Stays private by default — the admin decides who can use it and when.
  • Runs as a systemd service on your own Ubuntu server, with update and removal scripts.

Features

What you get

Broad link support

YouTube, youtu.be, Shorts, Instagram posts/Reels/profiles, and SoundCloud tracks.

Albums and captions

Multi-item Instagram posts are sent as media groups, with captions on the first file and a "Get full caption" button.

Admin controls

Activation, public/private mode, and forced channel subscription, all from inside Telegram.

Per-user cookies

Each allowed user can upload a personal cookies.txt for logged-in content, removable on demand.

Anti-spam guard

One active download and one new request every 5 seconds per user, with a live progress message.

/mp3 audio mode

Audio-only MP3 downloads on demand, with SoundCloud cover art sent first when available.

Bot commands

What the admin and users can run

/startStarts the bot; language is asked only once.
/mp3 <link>Sends audio only, as MP3.
/languageChanges the user's language at any time.
/cookiesExplains how to upload personal cookies.
/activateAdmin-only: enables downloads after install.
/forcejoin_on @channelAdmin-only: requires channel membership before downloading.

Getting it running

From clone to active bot

01

Run the installer

A one-line installer clones the repo and runs python3 install.py on Ubuntu, asking for your bot token and admin ID.

02

Activate from Telegram

Open the bot as the admin and send /activate — it downloads nothing until then.

03

Decide who can use it

Keep it private, open it to everyone, or require a channel join — all from the admin panel.
