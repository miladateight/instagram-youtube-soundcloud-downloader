What it does

One bot for every supported link.

The bot detects supported links automatically and turns them into the right Telegram delivery: video, photo, audio, or album.

01

Multi-platform downloads

Downloads YouTube, YouTube Shorts, Instagram posts, Reels, profiles and carousels, and SoundCloud tracks through yt-dlp.

02

Smart delivery

Sends video, photo, audio, or Telegram albums for multi-item posts, with captions on the first file and a 'Get full caption' button for long ones.

03

Admin-gated access

The bot stays inactive until the admin sends /activate, and can run private-by-default or public with forced channel subscription.

04

Per-user cookies

Each allowed user can upload and remove a personal cookies.txt for login-gated content, alongside an optional admin global cookies file.

05

Four-language interface

Persian, English, Arabic, and German, picked once on first start and changed anytime with /language.

06

Linux service install

A Python installer sets up a systemd service, with update, uninstall, and full-removal scripts and logs in logs/bot.log.

Flow

Send a link, get the media back.

step 1

Send a link

Paste a YouTube, Shorts, Instagram, or SoundCloud link, or use /mp3 for audio only.

step 2

The bot fetches it

yt-dlp resolves and downloads the media while a progress message moves from 0% to 100%.

step 3

Smart delivery

The result comes back as video, photo, audio, or an album, with a caption and cover art when available.

step 4

Admin stays in control

Activation, public or private mode, forced subscription, and cookies are all managed from inside Telegram.
